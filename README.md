## What is OOP?

Object Oriented Programming is a style of programming that differs from by imperative programming (the focus of term 1's work) by bundling data and methods into objects to model real-world entities, whereas imperative programming focuses on a sequential, step-by-step approach using commands to change the state.

Therefore we will look at OOP in terms of objects that are built into classes and objects and develop our coding skills aruond this.
